## Featured GenAI Project

# CortexFlow AI Workbench

A local-first AI engineering workbench that turns modern GenAI concepts into a working product-style system.

CortexFlow is built to show that AI engineering is not only prompt writing. It combines retrieval, context engineering, agents, tools, memory, evaluation, MCP-style interfaces, system design, and training data preparation.

| Area | Details |
|---|---|
| RAG | Document ingestion, chunking, hybrid retrieval, citations, source-aware answers |
| Vector Search | Keyword, BM25, TF-IDF cosine, and hybrid ranking |
| AI Agents | Planner, policy gate, tool registry, memory, reflection, final answer |
| MCP-style Tools | JSON-RPC interface for tools, resources, and prompts |
| Context Engineering | Builds model context from goal, memory, evidence, and tool traces |
| Multi-Agent Workflow | Planner, researcher, architect, critic, and finalizer roles |
| LLM Evaluation | Groundedness, citation coverage, token F1, safety, expected term recall, latency |
| RL Feedback | Epsilon-greedy bandit for retrieval strategy tuning |
| Computer Use | Safe simulated screen/browser action planning |
| AI Coding Workflow | Feature planning, test strategy, and code review helper |
| ML System Design | Blueprint generator for AI system architecture and metrics |
| Fine-Tuning Prep | SFT JSONL and preference-pair export from traces |
| Product Surface | FastAPI backend, Streamlit UI, Docker, tests, docs |
